# What's this?

my dotfile.


# How to use.

Please execute a below command.

```console
$ ./setup.sh
```

This script create symbolic links for dotfiles as followings:

* $HOME/.zshrc
* $HOME/.bash_profile
* $HOME/.bashrc
* $HOME/.emacs
* $HOME/.screenrc
* $HOME/.irbrc
* and etc...
